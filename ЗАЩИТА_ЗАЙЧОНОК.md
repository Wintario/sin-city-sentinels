# 🛡️ Защищённый пользователь: зайчонок

## Статус: ВЕЧНЫЙ АДМИНИСТРАТОР

Пользователь **`зайчонок`** имеет уникальный статус вечного администратора с абсолютной защитой от удаления.

---

## ⚠️ Правило защиты

> **пользователь зайчонок имеет вечные права администратора и не может быть удалён НИ ПРИ КАКИХ ОБСТОЯТЕЛЬСТВАХ.**
> 
> Это правило зафиксировано в коде и имеет наивысший приоритет для ВСЕХ администраторов без исключений.

---

## 🔒 Уровни защиты

Защита реализована на двух уровнях:

### 1. Уровень контроллера (`usersController.js`)

```javascript
// ⚠️ АБСОЛЮТНАЯ ЗАЩИТА: пользователь зайчонок не может быть удалён
if (user.username === 'зайчонок') {
  return res.status(403).json({ 
    error: 'Пользователь зайчонок имеет вечные права администратора и не может быть удалён',
    protection: 'permanent_admin_protection',
    message: 'Это правило зафиксировано в коде и не подлежит изменению'
  });
}
```

**Ответ при попытке удаления:**
- HTTP Status: `403 Forbidden`
- Сообщение: явное указание на защиту

### 2. Уровень модели (`userBan.js`)

```javascript
// ⚠️ АБСОЛЮТНАЯ ЗАЩИТА: пользователь зайчонок не может быть удалён
if (user && user.username === 'зайчонок') {
  throw new Error('ЗАЩИЩЁННЫЙ ПОЛЬЗОВАТЕЛЬ: зайчонок имеет вечные права администратора и не может быть удалён. Это правило зафиксировано в коде.');
}
```

**Защита на уровне БД:** выбрасывание ошибки перед любой операцией удаления.
